Duolca App Documentation

Duolca is NokiaEDU's portal to allow instructors to launch cloud VMs in Microsoft Azure. Find resources and documentation to help you run and build Duolca here.

What is Duolca?

Designed for instructors in NokiaEDU, Duolca is a portal to the Microsoft Azure Cloud Services Platform. It allows instructors to easily deploy their VMs.

How Do I Use It?

Deploy + Manage a Course
  1. Check the email sent you by NokiaEDU: contains COURSE NAME, DEPLOYMENT NAME, RESOURCE GROUP, and LOCATION NAME
  2. Input that information in the form on the home page, and click Deploy vLab
  3. Wait for about 10 minutes, before returning to the app and refreshing the index page:
    4. here
  4. Reenter the same information and click the Manage VLab button to view the Public IP Address!
